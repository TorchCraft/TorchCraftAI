78 /**
79  * This sampler expects as input an unordered_map<string, Variant>, containing
80  * an entry policyKey, which is a tensor of size [b, n]. It outputs the same
81  * map, with a new key "action", a tensor of size [b] where each entry action[i]
82  * is sampled from a normal distribution centered in policy[i]. It also expects
83  * the stdKey to be set, it will be used as the standard deviation of the
84  * normal. It can be either a float/double, in which case the deviation will be
85  * the same for the batch, or it can be the same shape as the policy, for a
86  * finer control. It also adds a key pActionKey which corresponds to the
87  * probability of the sampled action
88  */
