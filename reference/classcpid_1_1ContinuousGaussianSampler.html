Detailed Description

This sampler expects as input an unordered_map<string, Variant>, containing an entry policyKey, which is a tensor of size [b, n].

It outputs the same map, with a new key "action", a tensor of size [b] where each entry action[i] is sampled from a normal distribution centered in policy[i]. It also expects the stdKey to be set, it will be used as the standard deviation of the normal. It can be either a float/double, in which case the deviation will be the same for the batch, or it can be the same shape as the policy, for a finer control. It also adds a key pActionKey which corresponds to the probability of the sampled action
