20 /**
21  * Determines positions for buildings.
22  *
23  * For buildings that require a worker to build them, BuilderModule requires the
24  * UPC to specify a Dirac position. This module's job is to determine suitable
25  * positions based on an existing distribution over positions and various
26  * heuristics (via builderhelpers).
27  *
28  * Optionally, BuildingPlacerModel can be used for building placement. The model
29  * location is specified via the -bp_model command-line flag; if a valid model
30  * is found at the specified location, it will be loaded and used. If GPU
31  * support is available, the model will be run on the GPU.
32  *
33  * ProxyTasks are used to track execution of the downstream production task. If
34  * the production task will fail (e.g. because the location became unbuildable),
35  * retries will be attempted until the ProxyTask is cancelled (e.g. from an
36  * upstream module).
37  *
38  * This module will also reserve build tiles as unbuildable via TilesInfo.
39  */
