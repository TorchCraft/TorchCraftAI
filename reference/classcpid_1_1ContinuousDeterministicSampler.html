Detailed Description

This sampler expects as input an unordered_map<string, Variant> containing an entry policyKey, which is a tensor of size [b, n].

It outputs the same map, with a new key "action", a clone of the policy. It also adds a key pActionKey which corresponds to the probability of the sampled action (always 1 in this case)
